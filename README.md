# sp-tsm

This repository contains a proof-of-concept implementation of a key management system and trusted software module for an SP architecture.

To compile and run test scripts:
```
$ make
$ ./testkeychain
$ ./testkeycrypto
$ ./testtsm
$ ./demo1_driver
```
